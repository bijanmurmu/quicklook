# 🔍 QuickLook – Universal Text & Link Extractor

A minimal web tool to extract:

- 🔗 Links
- 📧 Emails
- 📞 Phone Numbers
- 📄 Top Keywords

Paste any text, and instantly extract the essentials!

---

## 🚀 Features

- Copy buttons for each item
- Clean responsive UI
- No libraries or frameworks – just HTML, CSS, JS
